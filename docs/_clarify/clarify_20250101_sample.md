유튜브 채널에서 영상 정보(제목, 조회수, 댓글)를 자동으로 수집해서

## 💭 문제 본질 (5 Whys)

**Q1. 얼마나 자주?**
매일 아침 (1시간 소요)

**Q2. 왜 수동으로 하나?**
자동화할 방법을 몰랐음. API 활용 몰랐음.

**Q3. 이 작업이 없으면?**
채널 성장도 못 추적하고, 콘텐츠 전략도 못 세움.

**Q4. 궁극적 목표는?**
(답변) "내 채널이 잘 자라고 있는지 한눈에 보고 싶어요"

**Q5. 그게 왜 중요해?**
(답변) "영상 성과를 빠르게 파악해서, 부족한 부분을 개선하고 싶어요"
