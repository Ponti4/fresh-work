# 📌 실제 예시: SNS 댓글 분류

| 항목 | 내용 |
|------|------|
| 업무명 | SNS 댓글 분류 |
| 작성일 | 2025-01-06 |
| 작성자 | 김철수 |

### 📊 현황
- **빈도**: 매일 (주 5회)
- **월 소요시간**: 약 6.5시간
- **연간 절감**: 약 78시간

### ⚡ 핵심 불편함
매일 아침 1시간을 댓글 수동 분류에 소비하여 마케팅 전략/영상 편집 진행 불가

### 🎯 자동화할 기능
- 댓글을 3가지 카테고리(긍정/부정/일반)로 자동 분류
- 분류된 댓글을 각 폴더로 자동 정렬

### 🎁 기대 효과
**본업 집중** - 콘텐츠 기획과 영상 편집에 집중
- 월 6.5시간 확보 → 핵심 업무 투자 시간 증가
- 콘텐츠 품질 향상

### 📡 외부 연동
아니오 - Claude Code만으로 가능
