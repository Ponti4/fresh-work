# AI 활용 사례 작성 가이드라인

> 게시글 작성 시 주의사항과 마무리 단계 안내입니다.

---

## ✍️ 작성 시 주의사항

### 1. 코드 제외 원칙
- **비개발자 대상**이므로 코드 블록 사용 최소화
- 필요하다면 간단한 설정값 정도만
- 대신 **자연어 프롬프트는 적극 포함**

**좋은 예:**
```
> Claude, 주간 리포트를 자동으로 만들어줘.
> 매주 금요일 오후 5시 데이터를 기준으로 작성하고,
> 엑셀 형식으로 저장해줘.
```

**나쁜 예:**
```python
def generate_report():
    # 복잡한 코드...
```

---

### 2. 프롬프트는 포함

**실제 사용한 자연어 프롬프트**를 인용문(`>`)으로 포함:
- 독자가 바로 참고할 수 있게
- 수정 가능한 부분은 `[수정 필요]`로 표시

**예시:**
```
> 우리 회사 고객 데이터를 분석해서
> [고객 세그먼트]별로 구매 패턴을 정리해줘.
> 결과는 표 형식으로 보여줘.
```

---

### 3. 스크린샷 적극 권장

**필수 스크린샷 위치:**
1. **Before 상황**: 기존 방식의 불편함 시각화
2. **AI 협업 과정**: Claude와 대화하는 화면
3. **결과물**: 완성된 결과 전체 화면
4. **Before/After 비교**: 시각적 비교

**촬영 팁:**
- 민감한 정보는 모자이크 처리
- 화면 전체보다는 핵심 부분 확대
- 화질 선명하게

---

### 4. 구체적 숫자 사용

**나쁜 예:**
- "시간이 많이 절약됐어요"
- "효율이 좋아졌어요"
- "팀원들이 좋아했어요"

**좋은 예:**
- "주 2시간 → 15분으로 87% 단축"
- "실수율 30% → 거의 0%"
- "팀원 5명 중 3명이 바로 도입"

---

### 5. 실패 사례도 포함

**시행착오가 오히려 더 도움됩니다:**
- 어떤 프롬프트가 통하지 않았는지
- 막혔을 때 어떻게 돌파했는지
- 다음엔 뭘 다르게 할 건지

**예시:**
```
처음엔 "데이터 정리해줘"라고만 했더니
원하는 형식이 아니었어요.

두 번째는 기존 엑셀 파일을 첨부하고
"이 형식으로 정리해줘"라고 했더니 딱 맞았어요.
```

---

### 6. 복붙 가능한 프롬프트 템플릿화

**독자가 바로 사용할 수 있게:**

```
### 프롬프트: 주간 리포트 자동 생성

> [데이터 파일]을 읽어서
> [기간: 월요일~금요일] 동안의
> [항목: 판매량, 고객 수, 매출]을 정리해줘.
>
> 결과는 [엑셀/PDF/표] 형식으로 저장해줘.

💡 대괄호 안은 본인 상황에 맞게 수정하세요.
```

## 🖼️ 마무리: 이미지 추천

글 작성이 완료되면, 독자 이해를 돕기 위해 **최소 3개 이미지**를 추천합니다.

### 추천 방식

**섹션 이름 기준으로 안내:**
- ❌ "12번 줄에 이미지 추가" (코드 라인 X)
- ✅ "문제 상황 섹션에 이미지 추가" (섹션 이름 O)

### 예시 추천

```
📸 추천 이미지 위치:

1. "소개: 문제 상황" 섹션
   - 기존 방식의 불편함을 보여주는 스크린샷
   - 예: 수작업 엑셀 화면, 반복 작업 화면

2. "진행 방법: AI와 협업한 과정" 섹션
   - Claude와 대화하며 작업하는 터미널/브라우저 화면
   - 예: 프롬프트 입력 → 결과 출력 화면

3. "결과와 배운 점: 결과물" 섹션
   - 완성된 결과물 전체 화면
   - 예: 자동 생성된 리포트, 대시보드

4. "결과와 배운 점: Before/After" 섹션
   - Before와 After를 나란히 비교한 화면
   - 예: 소요 시간 비교, 작업 과정 비교
```

## 📤 게시판 업로드 안내

### VSCode에서 업로드하는 방법

```
1. AI_CASE_STUDY.md 파일 우클릭
   → "미리보기 열기" 선택
   (단축키: Cmd+Shift+V / Ctrl+Shift+V)

2. 미리보기 화면에서 전체 선택
   (단축키: Cmd+A / Ctrl+A)

3. gpters 게시판 작성 화면에 붙여넣기
   (단축키: Cmd+V / Ctrl+V)

4. 추천된 위치에 이미지 추가
   - 드래그 앤 드롭 or 이미지 업로드 버튼

5. 최종 확인 후 게시!
```

### 이미지 업로드 팁

**gpters 게시판:**
- 이미지 드래그 앤 드롭 지원
- 또는 "이미지 추가" 버튼 클릭

**권장 형식:**
- PNG, JPG (10MB 이하)
- 가로 1200px 이하 (모바일 고려)
