# Setup Workspace - 초기 설정

> 당신의 상황을 파악하고 맞춤형 자동화 경로를 만들기 위한 첫 단계입니다.

**예상 시간: 5-7분**

---

## 📋 Step 1-4: 당신의 상황 파악

### Step 1: 기본 정보 - 이름 & 직업

이 두 가지 정보로 시작하겠습니다.

**질문 1: 자신을 소개해주세요. (이름/닉네임)**
- 자유 입력

**질문 2: 현재 어떤 일을 하고 계세요?**
- 예: "마케팅 프리랜서", "콘텐츠 크리에이터", "데이터 분석가" 등

---

### Step 2: 반복 업무 파악

자동화하고 싶은 반복 업무를 3개 작성해주세요.

**질문 3: 현재 반복하는 업무를 우선순위 순서로 3개 작성하세요**
- 예시:
  1. SNS 댓글 정리 (매일 2시간)
  2. 이메일 분류 (주 5시간)
  3. 데이터 입력 (주 3시간)

---

### Step 3: 4주 목표 & 기술 수준

**질문 4: 기술 경험 수준은?**
- 코딩 경험 없음 (비개발자)
- 코딩 조금 경험 있음 (기초 수준)
- 코딩 어느 정도 경험 있음


---

## 📋 Step 5-7: 시스템 정보 감지 & 저장

### Step 5: 운영체제 선택

**질문 6: 사용 중인 OS는?**
- Windows (Windows 10/11)
- Mac (Intel 또는 Apple Silicon M1/M2/M3)
- Linux (Ubuntu, Fedora 등)

---


### Step 7: 프로필 자동 저장

수집된 정보를 루트 `CLAUDE.md`의 "당신의 프로필" 섹션에 자동으로 저장합니다:

```markdown
## 📋 당신의 프로필

- **이름**: [입력값]
- **직업/상황**: [입력값]
- **첫 자동화**: [반복 업무 1번]
- **4주 목표**: [선택값]
- **반복 업무 1**: [입력값]
- **반복 업무 2**: [입력값]
- **반복 업무 3**: [입력값]
- **OS**: [감지값 또는 입력값]
- **Python**: ✅ 설치됨 / ❌ 미설치
- **Setup 날짜**: [자동 기록]
- **완료된 자동화**: 0개
```

> **참고**: Python이 필요한 경우는 스터디에서 설명하고 설치할 예정입니다.

---

## 🎉 완료!

모든 정보가 `CLAUDE.md`에 저장되었습니다.

---

## 🚀 다음 단계

이제 첫 자동화를 시작합니다!

```bash
/clarify "반복 업무 설명"
```

예시:
```bash
/clarify "인스타그램 팔로워 댓글을 매일 2시간 걸려 정리하고 있어"
```

이 명령어로 자동화 방안을 함께 설계합니다.
