# Clarify Automations - 질문 흐름 상세 가이드

## Phase 1: 현상 파악 (Fact Check)

### Q1: 빈도 - "이 업무를 얼마나 자주 하시나요?"

**선택지:**
- 1. **매일** (주 5회 이상) → 월 20회 이상
- 2. **자주** (주 2-4회) → 월 8-16회
- 3. **가끔** (월 1-3회) → 월 1-3회
- 4. **드물게** (분기 1회 미만) → 월 1회 미만

**해석:**
- 1, 2번: ✅ 자동화 유망 (반복성 높음)
- 3번: ⚠️ 검토 필요 (반복성 낮음)
- 4번: ❌ 자동화 부담 (한 번씩 하는 게 나을 수 있음)

**다음 질문으로 진행하기 전:**
사용자 응답을 명시적으로 확인하고 반영
```
감사합니다! 매일 반복하는 업무이군요.
```

---

### Q2: 소요시간 - "한 번 할 때 보통 얼마나 걸리나요?"

**선택지:**
- 1. **5분 미만**
- 2. **30분 내**
- 3. **1시간 내**
- 4. **1시간 이상**

**해석 (Q1과 조합):**

| Q1\Q2 | 5분미만 | 30분내 | 1시간내 | 1시간이상 |
|-------|--------|--------|--------|----------|
| 매일 | 약 1.5h/월 | 약 10h/월 | 약 20h/월 | 약 20h+/월 |
| 자주 | 약 0.5h/월 | 약 5h/월 | 약 10h/월 | 약 10h+/월 |
| 가끔 | - | 약 1.5h/월 | 약 3h/월 | 약 3h+/월 |
| 드물게 | - | - | - | - |

**자동화 ROI 판단:**
- **월 5시간 이상** → 자동화 가치 높음 ✅
- **월 1-5시간** → 조건부 (Q3 규칙성 중요) ⚠️
- **월 1시간 미만** → 자동화 비추 ❌

---

### Q3: 규칙성 - "매번 거의 동일한 방식으로 하시나요?"

**선택지:**
- 예 / 아니오

**해석:**

**예 (규칙성 높음):**
- 자동화 난이도: 낮음
- 성공 확률: 높음
- 예: SNS 댓글 수집 → 매번 같은 방식 ✅

**아니오 (규칙성 낮음):**
- 자동화 난이도: 높음
- 성공 확률: 중간
- 예: 클라이언트 요청 대응 → 매번 다름 ⚠️
- 단: Q4 극단화 분석으로 핵심만 자동화 가능

**Q1-Q3 종합 판단:**
```
매일 + 30분 이상 + 규칙성 높음 = 자동화 강력 추천 ✅✅✅
매일 + 5-30분 + 규칙성 높음 = 자동화 추천 ✅✅
자주 + 30분 이상 = 조건부 추천 ✅
가끔 + 1시간 이상 = 매우 조건부 ⚠️
```

## Phase 2: 극단화 (Expansion) - 문제의 본질

### Q4: 극단화 - "만약 이 업무가 내일부터 10배로 늘어난다면, 무엇이 가장 큰 문제일까요?"

**선택지:**
- 1. **시간 부족** → 양적 증가로 감당 불가
- 2. **실수 폭발** → 질적 저하 (오류 증가)
- 3. **멘탈 붕괴** → 심리적 부담
- 4. **포기하고 싶음** → 동기 상실

**해석 (문제의 본질):**

| 선택 | 핵심 문제 | 자동화 초점 |
|------|----------|-----------|
| **1. 시간 부족** | 반복으로 인한 시간 낭비 | ⏱️ 속도 자동화 (핵심) |
| **2. 실수 폭발** | 수작업의 휴먼 에러 | ✓ 정확성 자동화 (중요) |
| **3. 멘탈 붕괴** | 반복으로 인한 스트레스 | 🎯 심리적 부담 경감 |
| **4. 포기하고 싶음** | 동기 부족, 우선순위 낮음 | ⚠️ 자동화 효과 재검토 필요 |

**Q5로 진행하기 전 필수 확인:**
- Q1-Q3 + Q4 조합으로 자동화 필요성 최종 검증
- Q4에서 4번 선택 시 → Q5 이후 "정말 필요한가?" 재확인

## Phase 3: 가치 도출 (Value)

### Q5: 목표 - "이 문제가 완벽히 해결되어 여유 시간이 생긴다면, 무엇을 하시겠습니까?"

**선택지:**
- 1. **휴식** → 개인시간 확보
- 2. **자기계발** → 스킬 향상
- 3. **본업 집중** → 핵심 업무 투자
- 4. **칼퇴** → 워라밸

**해석 (가드레일):**

| 선택 | 의미 | 가드레일 신호 |
|------|------|------------|
| **1. 휴식** | 피로도 높음 | 긍정적 - 건강상 중요 ✅ |
| **2. 자기계발** | 성장 동기 | 매우 긍정적 - 장기 효과 ✅✅ |
| **3. 본업 집중** | 매출/영향력 증대 | 최고 긍정적 - ROI 명확 ✅✅✅ |
| **4. 칼퇴** | 워라밸 중시 | 긍정적 - 삶의 질 ✅ |

**Q5 역할:**
- 자동화의 **최종 가드레일**
- "정말 이 시간이 필요한가?" 재확인
- 4주 목표 연결 (CLAUDE.md의 프로필과 연계)

## 📊 질문별 월 소요시간 자동 계산

```
기본 계산식:

빈도별 월 회수:
- 매일 → 20회 (주5일 기준)
- 자주 → 10회 (평균)
- 가끔 → 2회 (평균)
- 드물게 → 0.5회

한 번 소요시간:
- 5분미만 → 5분
- 30분내 → 20분 (평균)
- 1시간내 → 40분 (평균)
- 1시간이상 → 90분 (평균)

월 소요시간 = 월 회수 × 한 번 소요시간
```

**예시:**
```
Q1: 매일 (20회/월)
Q2: 30분내 (20분)
→ 월 소요시간 = 20회 × 20분 = 400분 ≈ 6.5시간
```

## 💡 면접자(AI) 입장에서의 유의사항

### 언어 톤
- 비개발자 대상 → 쿠션어 사용
- "혹시", "혹은", "괜찮으시면" 등
- 선택지 설명은 자세하게 (예: "주 5회 이상" 명시)

### 응답 처리
- 각 답변 후 명시적 확인
  ```
  감사합니다! 매일 하시는 업무이군요.
  ```
- 사용자의 상황을 반영한 표현

### 질문 순서
- **절대 건너뛰기 금지** (Q1→Q2→Q3→Q4→Q5 순서 유지)
- 답변 대기 후 다음 질문
- 한 번에 하나의 질문만

### 최종 정리
- Q1-Q5 모든 답변 요약
- 월 소요시간 자동 계산
- Task 정의서 생성
- `docs/_clarify/` 경로에 저장 후 완료
