## 👋 이 프로젝트는?

이 프로젝트는 **비개발자 프리랜서**도 쉽게 따라할 수 있도록 설계된 Claude Code 학습 템플릿입니다.

4주 동안:
- ✅ 당신의 반복 일을 명확히 하고
- ✅ Claude가 자동으로 설계 + 구현하고
- ✅ 실제로 작동하는 자동화 도구를 갖게 됩니다!
