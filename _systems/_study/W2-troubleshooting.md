# 🆘 W2-W3: 문제 해결 가이드

**Plan Mode 진행 중 막혔을 때 읽기**

---

## 🔴 자주 나는 문제

### 1️⃣ "Plan Mode가 진행 중인데 시간이 오래 걸려요"
```
상황: /clarify 후 Plan Mode가 설계/구현 중
해결:
  → 5-10분 기다리세요 (정상)
  → 복잡한 자동화면 20분까지 가능
  → 멈춘 것 같으면 Claude에게 상태 물어보기
```

### 2️⃣ "생성된 코드가 실행이 안 돼요"
```
해결 순서:
1. 오류 메시지 전체 복사
2. Claude에게 보내기:
   "이 오류가 나왔는데 뭐가 문제야?"
   [오류 메시지 붙여넣기]
3. Plan Mode가 디버깅 시작
```

### 3️⃣ "Python 라이브러리 설치 오류"
```
예: ModuleNotFoundError: No module named 'pandas'

해결:
  → Claude: "pandas 설치 방법을 알려줘"
  → 또는 직접: pip install pandas
  → 그 후 다시 코드 실행
```

### 4️⃣ "API 키 설정 문제"
```
예: OpenAI API 키 없음

해결:
  1. API 키 발급받기 (해당 서비스)
  2. `.env.local` 파일에 저장
  3. Claude에게: "API 키 어디에 넣어?"
```

### 5️⃣ "/git-commit이 안 돼요"
```
해결:
  1. 깃 설정 확인: git config --list
  2. [git-setup-and-usage.md](_systems/wiki/git-setup-and-usage.md) 참고
  3. 여전히 안 되면 Claude에게 물어보기
```

## 💡 권장사항

### ✅ 할 것
- 오류 메시지 전체 캡처하고 Claude에게 보내기
- 코드 실행 전에 준비 파일 확인하기 (음성파일, CSV 등)
- 막히면 바로 물어보기

### ❌ 하지 말 것
- 코드 직접 수정 시도 (Plan Mode가 처리하도록)
- 오류 메시지 일부만 보내기
- 묵묵히 기다리기 (물어봐도 괜찮아)
