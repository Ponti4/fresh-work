# 👤 W1: 당신은 어떤 타입? (간단 버전)

**3분 안에 자신의 타입 찾기**

---

## 🎯 3가지 타입

### 타입 1️⃣: **시간이 부족한 실무자**
```
특징: 매일 반복되는 작업이 많음
예시: 메일 정리, 파일 분류, 리포트 작성
목표: 1시간 이상 매일 절약하기
```
👉 **Week 2 추천 첫 자동화**: 일일 메일 정렬 또는 파일 정리

### 타입 2️⃣: **데이터/마케팅 담당자**
```
특징: 데이터 수집 & 분석이 주요 업무
예시: SNS 데이터 모으기, 리포트 생성
목표: 데이터 수집 자동화로 시간 절약
```
👉 **Week 2 추천 첫 자동화**: 웹 크롤링 또는 데이터 자동 정리

### 타입 3️⃣: **공부하면서 수입 만드는 사람**
```
특징: 새로운 기술 배우기 + 돈벌기 동시
예시: 음성→텍스트, 이미지 변환, API 활용
목표: 3개월 안에 팔 수 있는 Skill 만들기
```
👉 **Week 2 추천 첫 자동화**: 음성 파일 자동 텍스트화 또는 이미지 변환

## 💭 당신은?

**이 중 가장 가까운 타입을 선택하고:**

```
/clarify "나는 [타입번호]번 타입이고, [구체적 반복 일]을 자동화하고 싶어"
```

**예시:**
```
/clarify "나는 1번 타입이고, 매일 아침 이메일 정렬하는데 30분이 걸려"
/clarify "나는 2번 타입이고, SNS에서 경쟁사 글을 수집하는 게 너무 번거워"
/clarify "나는 3번 타입이고, 음성 파일을 텍스트로 바꾸는 Skill을 만들고 싶어"
```

**다음은 자동으로 진행됩니다:**
- Plan Mode 에이전트가 설계 + 구현
- 당신은 생성된 코드 확인만 하면 됨
- 문제 있으면 Claude에게 물어보기
